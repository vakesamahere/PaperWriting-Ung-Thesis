\section{引言}
\label{sec:introduction}

服务网络设计（SND）是确定服务计划——路线、时刻表和资源容量——以满足网络上运输需求的问题。它在交通规划和运筹学中占据核心地位，因为服务计划决定了运营成本结构和客户所体验的服务水平。基础的SND模型将该问题建模为时空网络上的多商品流（MCF）优化，其中节点表示特定时间点的位置，弧表示服务运营或空闲资源持有\citep{boland2017continuous}。由此产生的混合整数规划（MIP）必须同时确定运营哪些服务、何时运营以及分配哪种资源类型，使得SND成为实践中最具挑战性的组合优化问题之一。

经典SND公式的一个关键局限性在于假设需求是外生的——独立于服务计划而固定。在现实中，当旅客可以在一个或多个运营商提供的竞争行程中进行选择，或完全选择不出行时，需求变为内生的：它取决于服务计划本身。将旅客选择模型纳入SND会在供给和需求决策之间产生反馈回路。服务计划决定了可用行程的集合，进而决定了所捕获的需求，再进而影响服务计划的盈利能力。这种耦合使问题显著变难——可行域现在包含将运营变量和需求变量联系起来的选择一致性约束——但也使其对实际规划而言更加现实和有价值。

SND与旅客选择的整合在多个交通领域中自然出现。在航空排班中，航空公司联合优化机队分配、时刻表编排和航线规划，同时考虑旅客如何根据时刻便利性、中转质量和票价水平在可用行程中进行选择\citep{wei2020airline, yan2022choice}。在城市轨道和地铁运营中，公交机构设计列车频率和停站模式，而通勤者根据旅行时间、换乘次数和拥挤程度选择路线。在城际巴士和铁路服务中，运营商规划走廊的服务频率和发车时刻，旅客根据发车时间、旅行时长和票价在竞争运营商和交通方式之间进行选择。在所有这些场景中，核心建模挑战是相同的：时空网络上可行行程的组合爆炸，加上通过离散选择框架内生化需求的需要。

考虑旅客选择的SND的标准建模方法将问题表述为时空网络上嵌入离散选择模型的多商品流规划。规划时域被离散化为时间区间，服务弧在该离散化网络上定义。旅客需求通过广义吸引力模型（GAM）\citep{gallego2015general}的基于销售的线性规划（SBLP）线性化实现内生化，该方法在MIP中提供了选择概率的可处理线性表示。由此产生的公式同时优化服务运营（资源到弧的二元分配）和需求分配（受选择一致性约束支配的连续市场份额变量）。然而，该模型规模极大：即使对于中等规模的网络，可行行程的数量也可达数十万，且二元服务变量与连续选择变量之间的交互产生了稠密、病态的约束矩阵。

现有求解方法在应用于此类问题时面临根本性局限。动态离散化发现（DDD）由\citet{boland2017continuous}提出，通过从粗糙离散化开始并仅在需要时迭代细化来高效压缩时空网络，保证收敛到连续时间最优解。然而，DDD是为具有外生需求的货运和服务网络问题设计的；它缺乏处理将需求分配与服务计划耦合的基于行程的旅客选择行为的机制。分解方法已被应用于提高基于选择的航空排班设计模型的可处理性\citep{yan2022choice}。然而，这些方法主要设计用于产生高质量的启发式解，并未为本文研究的行程级、资源耦合公式提供精确的动态细化框架。对完整公式的直接MIP求解对于现实规模的实例是不可行的：大量的行程变量——可能达数十万到数百万——使商业求解器不堪重负，由于弱LP松弛和分数退化，求解器在分支定界过程中停滞。

本文提出了一个克服这些局限性的精确求解框架。我们的主要贡献如下：
\begin{enumerate}
    \item 我们开发了一种行程聚合技术，结合时间离散化，通过构造"超级行程"产生紧的下界。这些聚合行程消除了松弛网络中分数资源分配产生的虚假收入，得到的松弛比完整模型的标准LP松弛显著更紧。
    \item 我们设计了一种基于频率固定和行程割的用于上界生成的修复算法，从松弛下界解中重构可行整数解。这种结构感知方法利用松弛中嵌入的信息来产生高质量的可行时刻表。
    \item 我们提出了一种细化策略，通过检测和解决三类松弛伪影——时序违规、间隔不可行性和虚假收入——动态地向离散化网络添加时间点，保证有限收敛到全局最优。
    \item 我们在真实世界实例上验证了该框架，证明了具有近五十万行程变量的此前不可处理的问题可以被求解到可证明的最优性。
\end{enumerate}
